# Booking-Site

A full-stack web application designed to emulate platforms like Booking.com. It allows users to browse apartments, manage reservations, and handle property listings.

## Key Technical Features
* **Tech Stack:** Built with **Vue.js** and **Ionic** for a mobile-optimized frontend, backed by **Node.js.
* **Core Functionality:** User authentication, reactive search/filtering, and a seamless booking workflow.
* **Architecture:** Utilizes **TypeScript** and a modern component-based structure for scalability and clean code.
